# Sistema Acadêmico Este projeto é um sistema acadêmico básico desenvolvido para praticar TDD, Code Review e CI. ## Módulos 1. **Lançamento de Notas**: Valida e salva as notas 1UP e 2UP lançadas pelo professor. 2. **Cálculo de Média**: Calcula a média do aluno com base nas notas lançadas. 3. **Status do Aluno**: Determina o status do aluno (Aprovado, Reprovado ou Aguardando final) com base na média. ## Executando os Testes 1. Clone o repositório. 2. Instale as dependências com `npm install`. 3. Execute os testes com `npm test`. ## Integração Contínua Os testes são executados automaticamente em cada pull request e merge na branch principal usando o GitHub Actions.
